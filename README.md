# wallapy-search

Small Python project scaffolded with Poetry that uses `wallapy` to query Wallapop listings.

## What’s in this repo

- Modern Python packaging via Poetry (`pyproject.toml` + lock file)
- `src/` layout (importable package)
- A tiny runnable example you can expand into a CLI, bot, or service

## Requirements

- Python 3.12+ (as set in `pyproject.toml`)
- Poetry installed

## Quickstart

### 1) Install dependencies

```bash
poetry install
